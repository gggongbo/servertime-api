그럼 아예 방향을 바꿔서 @https://time.navyism.com/?host=www.kbanknow.com api에 요청을 보내서 요청 보낸 당시의 서버 시간을 받아오는 간단한 api 서버를 구성하고 싶어.

사용하는 프레임워크랑 언어는 nest.js + typescript를 쓰고 싶고 fetch를 이용해서 get 요청을 보내고, 보낸 html 값을 파싱해서 나오는 서버시간을 return해주는 간단한 api 서버를 만들고 싶어

pnpm을 꼭 사용해야함

오케이 그럼 pnpm add @nestjs/core @nestjs/common @nestjs/platform-express reflect-metadata rxjs 이거대로 설치하고 simple-requirement.txt에 있는 요구사항대로 nestjs + typescript 기반의 api 서버 만드는 것을 진행해줘

get 요청으로 원하는 url을 보내면 

https://time.navyism.com/?host=${parameter_url} 로 매핑해줘서 time.navyism의 html 결과물을 리턴 받은 다음에 파싱해서 서버시간 부분만 가져온 뒤에 res.json에 담아서 보내줘야 해! 이게 코어한 기능이야 말그대로 아주 간단한 api 서버야.

guard나 기타 구성 요소는 만들 필요가 없어, 그리고 먼저 만들기 전에 요구사항을 정리한 PRD.korean.md를 만들어주고, 관련 프로젝트 진행 사항을 정리한 README.md를 만들어주길 바라!
